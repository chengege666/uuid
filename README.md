## ✨ 核心功能
- 🚀 **单条生成**：一键生成符合 RFC4122 标准的 UUID
- 📦 **批量生成**：支持 1-20 条批量生成，编号与 UUID 自动对齐，排版整洁
- 📋 **一键复制**：支持复制单条/批量 UUID，复制结果带编号（批量），提示清晰
- 💻 **离线可用**：纯 HTML/CSS/JS 实现，无需后端/服务器，打开即可用
- 📱 **响应式布局**：适配手机、平板、电脑等多设备，风格清新简约
